Commandes Git

  • git status : voir lezs fichiers en attente d'ajout
  • git add fichier.html : ajout un fichier à l'index GIT
  • git add . : ajouter tous les fichiers modifiés à l'index GIT
  •  git commit -m "commentaire"
    : faire un commentaire
  •  git diff
    : voir les différence entre l'index GIT et le fichier
  •  git stage .
    : mettre à jour un index
  •  git config -l
    : Afficher tous les details 
  •  git commit -a -m
    : ajout du commit -a -m 
  •  git rm -f fichier.txt
    : supprimer un fichier indexé de GIT
  •  git reset HEAD
    : suppression du dernier ajout avant commit
  •  git reset HEAD^
    : supprimer du dernier commit
  •  git reset HEAD^^
    : supprimer du avant dernier commit
  •  git checkout HEAD^ fichier.txt
    : supprimer du dernier commit pour le fichier fichier.txt
  •  git log --oneline
    : afficher les logs sur une seule ligne
  •  git log -2
    : afficher les logs sur deux lignes
  •  git checkout 2c3f4567 index.html
    : revient à une version antérieure du fichier
  •  git checkout master index.html
    : annule le dernier checkout
  •  git remote add origin https://github.com/azitouni/ProjectGit.git
    : créer une liaison entre le dossier local et le dossier sur GitHub.com
  •  git pull
    : obtenir la derniere version sur serveur 
  •  git push origin master
    : push du projet sur le dosier GitHub.com
  •  git revert
    : revenir à la version précédente 
  •  git branch
    : voir tous les branches 
  •  git branch ma-branche
    : connexion à une branche
